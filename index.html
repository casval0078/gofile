Gofileの使い方

  1. Gofileにアクセスする
  2. File Manager を選択する
  3. 左から３つめの青色のアイコンを選択し、ファイルをアップロードする
  4. アップロード完了後、緑色のアイコンを選択し、Change Visibility を選択する
  5. Make Public を選択すると Private から Public に変更される
  6. × で Public Setting を閉じ、再度緑のアイコンを選択
  7. 出てきたURLをコピーし、Xやインスタ等でフォロワーにリンクを教える
    8. 　　　
  8. 3. の操作でファイルを追加可能なので、２回目以降は「ファイルを追加したよ！」のお知らせのみでＯＫ！
